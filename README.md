# NOCTURNE.

> **Engineering Premium Digital Experiences in the Dark.**

Nocturne is a high-performance digital engineering agency specializing in ultra-fast web applications, advanced mobile engineering, and custom AI tooling. This repository contains the official landing page, designed with a premium, ultra-dark aesthetic and motion-heavy user experience.

## ✨ Key Features

- **Premium Aesthetic**: Ultra-dark theme utilizing glassmorphism, glowing accents, and high-fidelity typography.
- **Cinematic Motion**: Custom-engineered scroll reveal animations and dynamic gradient flows using Framer-like precision.
- **Precision Responsive Design**: Fully optimized for every device, from ultra-wide monitors to small-screen mobile devices.
- **Aurora Background Engine**: Custom SVG-based animated background blobs for a modern, fluid atmosphere.

## 🛠️ Technology Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Animations**: Custom CSS Keyframes & Intersection Observer API
- **Fonts**: Inter (via Google Fonts)

## 🚀 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the results.

## 📦 Production Deployment

The project is optimized for production-ready performance. To build the application:

```bash
npm run build
```

The optimized bundle will be available in the `.next` directory.
